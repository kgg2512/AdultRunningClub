## 제품 개요

| 항목 | 내용 |
|------|------|
| 유형 | 럭셔리 러닝 멤버스클럽 앱 |
| 타겟 | 30-50대 프리미엄 러너 |
| 모델 | 멤버십 구독 |
| 핵심 가치 | 검증된 러너들의 프라이빗 커뮤니티 |

## 핵심 기능

- 멤버십 가입 및 인증
- 러닝 크루 매칭
- 이벤트 / 대회 정보
- 프리미엄 러닝 코스 공유

## 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (PWA)
- **Backend**: Supabase
- **인프라**: Cloudflare Workers
- **배포**: GitHub Pages

## 개발 현황

| Phase | 내용 | 상태 |
|-------|------|------|
| Phase 1 | 프로토타입 — 멤버십 가입 흐름 | ✅ 완료 |
| Phase 2 | 결제 연동 (Stripe) | ⏳ 예정 |
| Phase 3 | 앱 런칭 | ⏳ 예정 |
